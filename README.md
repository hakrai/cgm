# Chat Group Manager Chrome Extension

## How to use?

 - Clone this repo / [download zip](https://github.com/hakrai/cgm/archive/refs/heads/main.zip) and extract
 - Open [chrome://extensions](chrome://extensions) on right top corner enable Developer mode
 - On top left corner click Load Unpacked then browse the cloned dir and click open
 - Go to arena.ai if sidebar right not found refresh the page
